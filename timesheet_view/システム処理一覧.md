# timesheet_view システム処理一覧

## 対象範囲
- 対象: `src/` 配下の業務処理、状態遷移、データ取得/更新、モーダル操作、Dataverse 連携、エラーハンドリング
- 対象外: 純表示専用コンポーネント（`Button`/`Input`/`Select`/`Spinner` 等の描画のみ）、CSS、型定義のみのファイル

## トリガー分類
- `初期表示`: 画面起動時、モーダル起動時、`useEffect` 初回実行
- `ユーザー操作`: クリック、選択変更、検索、保存、削除など
- `状態同期`: props/state 変更に追従する自動処理
- `データ取得`: React Query や Dataverse API による取得処理
- `API内部`: Dataverse クライアント共通処理・CRUD 内部処理
